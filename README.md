# GetRandomParagraph
Get a random paragraph from a text file.

E.g:

#### --- document.txt ---
1. Hello
2. mi name is
3.
4.
5. Jesus
6.
--------------------

Result:
1. Hello
2. mi name is
3. Jesus
--------------------
--------------------

Output:
```
File name: document.txt
Number of lines: 3
Random line selected: 2
"mi name is"```
